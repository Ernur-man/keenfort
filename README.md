# KeenFort: Коммерческий сайт курсов английского языка

## 📚 Описание

Профессиональный сайт для онлайн-школы английского языка с полной функциональностью.

### Технологический стек
- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Интеграция**: Google Sheets API (чтение контента, запись заявок)
- **Security**: XSS защита, CSRF токены, валидация форм

## 🚀 Секции сайта

1. **Главная** - Привлекательная посадочная страница
2. **Обучение** - Описание курсов и программ
3. **Преподаватели** - Биографии и опыт преподавателей
4. **Отзывы** - Отзывы студентов из Google Sheets
5. **Тарифы** - Ценовые пакеты и условия
6. **Регистрация** - Форма записи на курсы (сохраняется в Google Sheets)

## 🔒 Безопасность

- ✅ Скрытие Google API токенов (backend-only)
- ✅ XSS защита через DOMPurify
- ✅ Валидация всех форм (Zod)
- ✅ CSRF токены
- ✅ Security headers

## 📦 Установка

```bash
npm install
```

## 🔑 Конфигурация Google Sheets

1. Создайте сервисный аккаунт в Google Cloud Console
2. Загрузите JSON ключ
3. Скопируйте значения в `.env.local`

## 🎮 Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)
